API - Projet


Utilisation de l'API pour les projets :

GET : /projects
Récupère tous les projets en base

GET : /projects/{id}
Récupère le projet en fonction de l'identifiant : id

POST : /projects
Paramètre : Modèle de donnée d'un Projet en JSON dans le body de la requête.
Insère un nouveau projet en base

PUT : /projects/{id}
Paramètre : Modèle de donnée d'un Projet (données du projet à mettre à jour) en JSON dans le body de la requête.
Met à jour totalement ou partiellement un projet en base en fonction de l'identifiant du projet : id

DELETE : /projects/{id}
Supprime un projet en base en fonction de l'identifiant du projet : id

Rappel d'utilisation : URL point d'entrée + URL méthode voulu
Exemple HTTP GET : http://localhost:3000/projects/
