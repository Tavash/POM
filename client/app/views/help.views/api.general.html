API RESTFul

Définition REST
Style d’architecture permettant de construire des applications orientées Web.
Ensemble de conventions et de bonnes pratiques à respecter.
L’architecture REST utilise les spécifications originelles du protocole HTTP.
Note d'information : Par soucis de simplification du projet, nous n'avons pas sécurisé l'API de POM par une authentification.

Technologies :
  • Serveur : NodeJS
  • Base de données : MongoDB

Pour les onglets suivant, le point d'entrée d'accès à l'API est : http://localhost:3000
Pour l'utiliser, il suffit de concaténer l'URL point d'entrée + URL méthode voulu
