Statistiques - Généralités


Avec POM, vous pouvez visualiser les graphiques statistiques pour l’analyse des données sur les projets ainsi que plusieurs indicateurs comme le dashboard en page d'acceuil.

Page d’accueil :

Dashboard : Les tuiles permettent d’afficher le nombre de tâches prioritaires, les "à venir", les terminées, les nouvelles tâches et celles annulées des projets qui sont "En cours".

Tous mes projets : Récapitulatif de tous vos projets avec des indications d'avancement dans le temps et budgétaire.

Toutes mes tâches : Récapitulatif de toutes vos tâches avec des indications d'avancement dans le temps et budgétaire.
