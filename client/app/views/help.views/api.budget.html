API - Budget


Utilisation de l'API pour les budgets :

GET : /budgets
Récupère tous les budgets

GET : /budgets/{id}
Récupère le budget avec l'identifiant : id

POST : /budgets
Paramètre : Modèle de donnée d'un Budget en JSON dans le body de la requête.
Insère un nouveau budget en base

DELETE : /budgets/{id}
Supprime un budget en base en fonction de l'identifiant du budget : id

Rappel d'utilisation : URL point d'entrée + URL méthode voulu
Exemple HTTP GET : http://localhost:3000/budgets/
