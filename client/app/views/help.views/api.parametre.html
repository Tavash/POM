API - Paramètres


Utilisation de l'API pour les paramètres :

http:localhost:3000/settings
Récupère tous les paramètres

http:localhost:3000/settings/roles
Récupère tous les rôles

http:localhost:3000/settings/fonctions
Récupère toutes les fonctions

http:localhost:3000/settings/statuts
Récupère tous les statuts

http:localhost:3000/settings/categories
Récupère toutes les catégories de tâche
