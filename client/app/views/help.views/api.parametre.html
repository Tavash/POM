API - Paramètres


Utilisation de l'API pour les paramètres :

GET : /settings
Récupère tous les paramètres

GET : /settings/roles
Récupère tous les rôles

GET : /settings/fonctions
Récupère toutes les fonctions

GET : /settings/statuts
Récupère tous les statuts

GET : /settings/categories
Récupère toutes les catégories de tâche

Rappel d'utilisation : URL point d'entrée + URL méthode voulu
Exemple HTTP GET : http://localhost:3000/settings/
