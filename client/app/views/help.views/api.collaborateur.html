API - Collaborateur


Utilisation de l'API pour les collaborateurs :

GET : /collaborators
Récupère tous les collaborateurs

GET : /collaborators/{id}
Récupère le collaborateur avec l'identifiant : id

GET : /collaborators/{id}/projects
Récupère tous les projets d'un collaborateur identifié par : id

POST : /collaborators
Paramètre : Modèle de donnée d'un Collaborateur en JSON dans le body de la requête.
Insère un nouveau collaborateur en base

PUT : /collaborators/{id}
Paramètre : Modèle de donnée d'un Collaborateur (données du projet à mettre à jour) en JSON dans le body de la requête.
Met à jour totalement ou partiellement un collaborateur en base en fonction de l'identifiant du collaborateur : id

DELETE : /collaborators/{id}
Supprime un collaborateur en base en fonction de l'identifiant du collaborateur : id

Rappel d'utilisation : URL point d'entrée + URL méthode voulu
Exemple HTTP GET : http://localhost:3000/collaborators/
